## 🔌 后端对接

### API 基础地址
```
http://localhost:8000
```

### 认证方式
```
Authorization: Bearer <access_token>
```

### 已实现的接口对接
1. ✅ POST /api/auth/send-code - 发送验证码
2. ✅ POST /api/auth/login - 登录
3. ✅ POST /api/auth/logout - 登出
4. ✅ GET /api/user/me - 获取用户信息
5. ✅ GET /api/voices - 获取音色列表
6. ✅ POST /api/synthesis - 创建合成任务
7. ✅ GET /api/history - 获取历史记录
8. ✅ DELETE /api/history/:id - 删除历史
9. ✅ GET /api/admin/credits/transactions - 获取流水
10. ✅ GET /api/admin/credits/export - 导出流水

```bash
npm install
```

### 2. 确保后端运行
确保后端服务在 `http://localhost:8000` 运行

### 3. 启动前端
```bash
npm run dev
```

### 4. 访问应用
```
http://localhost:3000
```
